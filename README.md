## 🧩 Installation
```
git clone https://github.com/t1ltxz-gxd/disgo-template.git 
cd disgo-template
go build
```

## ⚙ Configuration
Open `config/*.yml` and fill in the values

### Setup environment variables
rename `.env.sample` to `.env` and fill in the values

___

## 🌐 Deployment
Run `docker-compose up -d`.
